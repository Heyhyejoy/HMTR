## Setup and Installation

1. **Clone the repository**:
https://github.com/Heyhyejoy/HMTR.git

2. **Navigate to the project directory**:

        cd HMTR
        
       
3. **Install dependencies**:

        npm install

4. **Run the application**:

        npm start
